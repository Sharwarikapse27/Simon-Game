# Simon-Game
A fun and interactive Simon Says game built using HTML, CSS, and JavaScript. Test your memory by repeating color sequences with increasing difficulty levels.
Simon Says Game

#Features
1. Three difficulty levels: Easy, Intermediate, Expert
2. Progressive level-based gameplay
3. High score tracking using Local Storage
4. Restart and Home navigation options
5. Clean UI with smooth animations

#Technologies
  HTML5
  CSS3
  JavaScript (Vanilla)

#How to Play
1. Click Start Game
2. Select a difficulty level
3. Observe the color sequene
4. Repeat the sequence correctly to advance
5. The game ends on an incorrect input
